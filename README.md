## プロジェクトの概要

このアプリケーションは、**CLIP (Contrastive Language-Image Pre-training)** モデルを利用して、アップロードされた複数の画像からユーザーの好みを分析します。画像に重み付けをすることで、特定のアイテムやスタイルをより重視した分析が可能です。

分析結果は以下の3つの観点から提供されます。

1. **フォーマル度/カジュアル度**: アップロードされた画像の全体的なフォーマルさを評価します。

2. **スタイル分析**: ストリート系、ヴィンテージ、ミニマリストなどの系統や、色、トーンなどの属性を分析します。

3. **アイテム提案**: データベースから、最もユーザーの好みに近い画像を提案します。

## 必要なファイル

このアプリケーションは2つのPythonファイルで構成されています。

* `pre_app.py`: アプリケーションで利用するデータベースを生成するための前処理スクリプトです。

* `st_app.py`: ユーザーインターフェースを備えたStreamlitアプリケーション本体です。

## セットアップと実行方法

### ステップ1: データベースの準備

まず、`pre_app.py`を実行して、アプリケーションが使用する画像の特徴量ベクトルと画像データを生成します。

```
python pre_app.py


```

このスクリプトは、`ashraq/fashion-product-images-small`データセットの画像10000枚をダウンロードし、`features.pt`と`images.pkl`という2つのファイルをカレントディレクトリに保存します。これらのファイルは`st_app.py`の実行に必須です。

### ステップ2: アプリケーションの起動

データベースの準備が完了したら、Streamlitを使ってアプリケーションを起動します。

```
streamlit run st_app.py


```

コマンド実行後、ブラウザでアプリケーションが立ち上がります。

* **Python**: プロジェクト全体のプログラミング言語

* **Hugging Face Transformers**: CLIPモデルの利用

* **Hugging Face Datasets**: データセットの読み込み
